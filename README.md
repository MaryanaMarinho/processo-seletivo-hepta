# Projeto para processo seletivo da Hepta

## Projeto clonado apartir <https://gitlab.com/gustavo-oliveira/processo-seletivo>

#### Back-end 

* Java
* REST com Jersey
* Banco de dados MySQL

#### Front-end

* Vue.js
* Bootstrap

## Instruções

* Clone o projeto 
* Importe o projeto no Eclipse
* Faça o deploy em um servidor de aplicação
* Acesse o link http://localhost:8080/mercado/ para visualizar o projeto.
